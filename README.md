# mxcro-frontend

Frontend for [mxcro](https://github.com/jan146/mxcro), made with Vue.js, TypeScript and Bun.

## Installation & configuration

```sh
bun install
```

### Compile and Hot-Reload for Development

```sh
bun dev
```

### Type-Check, Compile and Minify for Production

```sh
bun run build
```
